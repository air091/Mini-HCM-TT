# Mini-HCM-TT

WHAT IS THE PROJECT?
Project is to make a Mini Human Capital Management, specifically in Time tracking. The project will be built in Node.js (Express), Firebase (Firestore), and React.js.

WHAT IS THE MINIMUM VIABLE PRODUCT?
The minimum viable product of this project is to make employees calculate their time in and out in work. Users punch in and out with time, and admins can view and edit their punches.
