# Mini-HCM-TT

# WHAT IS THE PROJECT?
Project is to make a Mini Human Capital Management, specifically in Time tracking.
The project will be built in Node.js (Express), Firebase (Firestore), and React.js.

# WHAT IS THE MINIMUM VIABLE PRODUCT?
The minimum viable product of this project is where employees can record working hours through punch in/out, and admins can manage and review all time records.
